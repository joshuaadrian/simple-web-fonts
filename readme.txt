== Description ==

What this plugin does is retrieve a twitter / instagram feeds and places them in a cache file. This is feeds are then retrieved by placing shortcodes on your pages, posts, or widgets

**Features:**

* Following IDs can be revealed: Posts, Pages, Categories, Links, Media, Users, Comments, Link categories, Tags (WP 2.8 and above)
* Each ID can be de-/activated seperately, plus you can **allow/permit user roles to see the IDs**

**Included languages:**

* English

== Installation ==

1. Download the plugin and unzip it.
2. Upload the folder social-feeds/ to your /wp-content/plugins/ folder.
3. Activate the plugin from your WordPress admin panel.
4. Installation finished.
